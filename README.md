# OCR Application for Windows 10 or later

## Overview
This application extracts text from images.
using [Windows.Media.Ocr](https://learn.microsoft.com/en-us/uwp/api/windows.media.ocr?view=winrt-22621)

## How to use
Capture with Windows Snipping Tool  
Paste into OCRBOY  
Text will be extracted  

## Notes
OCRBOY works on Windows 10 or later. OCR results are not perfect.


# Windows10以降用のOCRアプリケーション


## 概要
画像からテキストを抽出するアプリケーションです。
[Windows.Media.Ocr](https://learn.microsoft.com/en-us/uwp/api/windows.media.ocr?view=winrt-22621)を使用しています。


## 使い方
WindowsのSnipping Toolでキャプチャ  
OCRBOYにペースト  
テキストが抽出されます


## 注意点
OCRBOYはWindows10以降で動作します。
OCR結果は完全ではありません。
